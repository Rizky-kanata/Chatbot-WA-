# EIGER Welcome Page

Folder ini khusus untuk deploy welcome page ke Vercel.

## Deploy Vercel

Pilih folder ini sebagai root project:

```text
landing-page
```

Pengaturan Vercel:

```text
Framework Preset: Other
Build Command: kosongkan
Output Directory: ./
Install Command: kosongkan
```

Nomor WhatsApp dan nama anggota diatur di `welcome-config.js`.
